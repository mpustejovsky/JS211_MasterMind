Instructions

Select the colors you think will match the code the computer selected. Then click the button to lock in your guess. Information about your guess will show up in the message center. Also a red peg will be placed below the board for each item you guessed with correct color and correct location. White markers will be placed for each color you selected that matches, but are not in the right location. The placement of the pegs do not corrolate to the gameboard and the pegs can be placed in any position. They just tell you the number in correct locations and the number of others with right color but wrong location. You can cheat after the first guess and the seceret code will be displayed for a short time. However cheaters never win and winners never cheat.
